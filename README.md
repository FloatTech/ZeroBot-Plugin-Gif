# ZeroBot-Plugin-Gif
[ZeroBot QQ机器人](https://github.com/wdvxdr1123/ZeroBot)插件，可以制作各种沙雕gif图
>部分必须下载素材包使用:https://github.com/tdf1939/sucai


触发方式分3种：
1. [指令词]+[qq号] 如：爬123456
2. [指令词]+[图片] 如：爬[图片]
3. [指令词]+[艾特] 如：爬@小H
