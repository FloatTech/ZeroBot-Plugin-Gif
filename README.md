# ZeroBot-Plugin-Gif
ZeroBot插件，可以制作各种沙雕gif图
触发方式分3钟：
1. [指令词]+[qq号] 如：爬123456
2. [指令词]+[图片] 如：爬[图片]
3. [指令词]+[艾特] 如：爬@小H
